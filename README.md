# css-utils
Set of CSS utility classes for repeatedly writing basic styles.

Inspired by [Medium_CSS_Style_Guide](https://gist.github.com/cuibonobo/16f555c0047ab80044cf).

# Getting Started
It's there on build directory, just copy and use.

# Classes

###### position
- `.u-positionRelative`
- `.u-positionAbsolute`

###### display
- `.u-displayNone`
- `.u-displayBlock`
- `.u-displayInlineBlock`
- `.u-displayInline`
- `.u-displayListItem`
- `.u-displayTable`
- `.u-displayTableRow`
- `.u-displayTableCell`

###### text-align
- `.u-texAlignLeft`
- `.u-textAlignRight`
- `.u-textAlignCenter`

###### vertical-align
- `.u-verticalAlignBaseline`
- `.u-verticalAlignBottom`
- `.u-verticalAlignMiddle`
- `.u-verticalAlignTop`

###### float
- `.u-floatLeft`
- `.u-floatRight`
